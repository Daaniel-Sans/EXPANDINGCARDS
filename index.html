01

¿Qué es esto?

Este componente es un conjunto de tarjetas expandibles construidas con HTML, CSS y JavaScript puro. Al hacer clic en cualquiera de ellas, se expande suavemente revelando su contenido mientras las demás se contraen, creando una experiencia de navegación limpia e intuitiva.

02

¿Cómo funciona?

Cada tarjeta escucha un evento de clic con JavaScript. Al activarse, se añade una clase active que modifica el valor de flex y la opacidad mediante transiciones CSS. Cuando se activa una nueva tarjeta, la anterior pierde su clase y vuelve a su estado original.

03

¿Qué técnicas usa?

El proyecto combina flexbox para distribuir las tarjetas en el layout, transiciones CSS para animar la expansión y el cambio de color, y manipulación del DOM con querySelector y classList para gestionar el estado activo de cada tarjeta en cada momento.

04

¿Qué he aprendido?

Construir este componente ha servido para entender cómo combinar CSS y JavaScript para crear interfaces interactivas sin librerías externas. También ha puesto en práctica conceptos como la gestión de estados, las transiciones encadenadas y la importancia de separar lógica visual y comportamiento.
